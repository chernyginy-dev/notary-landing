# Notary Services Landing Page

A modern bilingual landing page website designed for notary services.  
The project focuses on a clean, professional design with a minimal layout and smooth UI animations.

## 🌐 Live Demo

https://chernyginy-dev.github.io/notary-services-landing

## ✨ Features

- Modern minimalist landing page design
- Bilingual structure (Ukrainian / English)
- Fully responsive layout
- Mobile navigation with animated slide-in menu
- Smooth scroll and hover animations
- Clean professional aesthetic suitable for legal services
- Contact section with quick communication links

## 🧱 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

No frameworks or libraries were used.
